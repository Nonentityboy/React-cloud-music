# React-cloud-music
 React 全家桶 (包含 hooks)  immutable 移动端音乐 WebApp

访问地址：http://148.70.47.125:9002


## 项目技术栈：

### 前端部分：

* `react v16.8 全家桶 (react，react-router)`
* `redex`
* `redux-thunk`
* `immutable`
* `react-lazyload`
* `styled-components`
* `axios`

### 后端部分：

github上开源的 NodeJS 版 api [接口](https://github.com/Binaryify/NeteaseCloudMusicApi)
